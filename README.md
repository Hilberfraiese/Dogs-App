## Info de la App

Esta App permite:
- Ver todos los perros que contiene la API, a travez del paginado.
- Filtrar por: Nombre, Raza y Temperamentos.
- Ordenar por: Peso, Altura y Alfabetico.
- Ver detalles de cada perro.
- Crear tu perro.

## Tecnologias utilizadas:
- Javascript.
- Node.js.
- React.
- React-Redux.
- Express.
- Sequelize.
- PostgresSQL.
- ECMAS6.
- CSS.
- Material UI.
- Axios.
- React Hooks.
